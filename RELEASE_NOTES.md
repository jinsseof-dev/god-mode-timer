# v1.22 - 글로벌화 및 통계 고도화

v1.22 업데이트는 God-Mode Timer의 글로벌화를 위한 첫 걸음입니다. 이제 한국어뿐만 아니라 영어, 일본어, 중국어로도 갓생타이머를 즐길 수 있으며, 더욱 강력해진 통계 기능을 제공합니다.

## 🌐 새로운 기능 (New Features)

*   **다국어 지원 (Multi-language Support)**:
    *   **한국어(Korean)**, **영어(English)**, **일본어(Japanese)**, **중국어(Chinese)**를 정식 지원합니다.
    *   설정(Settings) 메뉴에서 원하는 언어로 즉시 변경할 수 있습니다.
    *   앱 최초 실행 시 운영체제의 언어 설정을 감지하여 자동으로 언어를 적용합니다.
*   **통계 기능 강화 (Enhanced Statistics)**:
    *   **시간대별 분석 (Hourly Analysis)**: 하루 중 언제 가장 집중했는지 시간대별로 파악할 수 있습니다.
    *   **로그 관리**: 잘못 기록된 로그를 수정하거나 삭제할 수 있는 기능을 추가했습니다.
    *   **디자인 개선**: 그래프 툴팁 디자인을 개선하여 가독성을 높였습니다.
*   **데이터 관리 (Data Management)**:
    *   **데이터 저장 위치 변경**: 데이터가 '내 문서' 폴더에 저장되어 앱 삭제 후에도 보존됩니다.
    *   **데이터 초기화 (Reset)**: 설정 창에서 모든 기록을 초기화할 수 있는 기능을 추가했습니다.
    *   **자동 마이그레이션 (Auto Migration)**: 기존 텍스트 파일로 저장되던 로그를 SQLite 데이터베이스로 자동 이관하여 안정성과 성능을 개선했습니다.

## 🛠️ 개선 사항 (Improvements)

*   **콘솔 창 숨김**: 윈도우 환경에서 실행 시 불필요하게 뜨던 콘솔 창(Command Prompt)을 제거하여 더 깔끔하게 실행됩니다.
*   **UI 텍스트 전면 개편**: 메인 화면뿐만 아니라 팝업, 통계, 후원 화면 등 앱 내 모든 텍스트를 다국어 처리하여 일관성을 높였습니다.
*   **폴더 구조 최적화**: 언어 리소스 파일들을 체계적으로 관리하도록 내부 구조를 개선했습니다.
*   **종료 경험 개선**: 앱 종료 시 매번 뜨던 확인 팝업을 제거하여 더 빠르게 종료할 수 있습니다. (단, 집중 모드 중에는 실수 방지를 위해 유지됩니다)
*   **통계 인터랙션 개선**: 날짜 헤더의 화살표를 누르면 목록을 접고, 날짜 텍스트를 누르면 해당 날짜의 파이 차트를 보도록 동작을 분리했습니다.

# v1.20 - UI/UX 고도화 및 테마 지원

이번 v1.20 업데이트에서는 사용자 경험(UX)을 대폭 개선하고, 다양한 모니터 환경에 대응하기 위한 UI 크기 조절 기능을 추가했습니다.

## ✨ 새로운 기능 (New Features)

*   **🎨 테마 시스템 도입**: 라이트 모드와 다크 모드를 지원하며, 설정 창에서 실시간으로 미리볼 수 있습니다.
*   **🖥️ UI 크기 조절 (Scaling)**: 설정 창에서 50% ~ 200%까지 UI 배율을 자유롭게 조절할 수 있습니다. 4K 모니터나 노트북 등 다양한 환경에서 최적의 크기로 사용하세요.
*   **⚙️ 설정 창 레이아웃 개편**: 설정 항목이 많아짐에 따라 가로형 2단 레이아웃으로 변경하여 시인성을 높였습니다.
*   **📊 반응형 통계 창**: 통계 창의 크기를 조절하면 내부 그래프와 글자 크기도 함께 최적화됩니다.

## 🛠️ 개선 사항 (Improvements)

*   **고해상도 아이콘 적용**: 작업 표시줄과 Alt+Tab 화면에서 흐릿하게 보이던 아이콘을 256px 고해상도 버전으로 교체했습니다.
*   **할 일 입력창 시인성 확보**: 창 크기가 작을 때 할 일 입력창이 가려지는 문제를 해결했습니다.
*   **팝업 디자인 개선**: 종료 확인, 설정 저장 등 모든 팝업 창의 텍스트와 버튼을 중앙 정렬하여 깔끔하게 다듬었습니다.
*   **버전 정보 표시**: 설정 창 하단에 현재 앱 버전(v1.20)이 표시됩니다.

## 🐛 버그 수정 및 기타 (Fixes & Misc)

*   **MSIX 인코딩 수정**: 설치 화면에서 한글이 깨져 보이는 문제를 해결했습니다.
*   **빌드 시스템 개선**: `.env` 파일의 버전 정보를 빌드 프로세스 전반에 자동 적용하도록 개선했습니다.
*   **시스템 트레이 제거**: 앱의 복잡도를 줄이고 사용성을 명확히 하기 위해 트레이 최소화 기능을 제거했습니다.
